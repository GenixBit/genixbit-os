GenixBit package infrastructure is being prepared.

This domain will publish signed GenixBit OS packages, repository metadata, public verification keys, release channels, and package indexes after the signing, snapshot, promotion, rollback, and recovery processes pass security review.

No production APT repository is active on this preview. Do not add this domain to an APT source until an official GenixBit release announcement provides the exact signed configuration.
