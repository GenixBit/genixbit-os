# GenixBit Approved Logo Source Bundle

Use these files as the approved visual references for GenixBit OS branding.

## Files

- `branding/source/genixbit-approved-square.jpg`
  - Approved GB monogram with GENIXBIT wordmark.
- `branding/source/genixbit-approved-horizontal.png`
  - Approved horizontal GB + GENIXBIT lockup with tagline:
    `Innovate. Integrate. Illuminate`
- `branding/source/genixbit-3d-reference.png`
  - Premium 3D reference for wallpapers, installer welcome artwork,
    website hero sections, and release graphics.

## Production rules

1. Preserve the exact GB monogram geometry.
2. Do not replace it with an infinity/orbital placeholder.
3. Create flat vector variants for small system UI surfaces.
4. Use the 3D reference only for large artwork.
5. Keep the tagline exactly:
   `Innovate. Integrate. Illuminate`
6. Do not distort, stretch, rotate, or redraw the GB mark into a different symbol.
7. Generated SVGs must be visually compared with these approved references.
8. Keep these source files in the repository only when GenixBit owns the rights
   and intentionally approves public redistribution.
