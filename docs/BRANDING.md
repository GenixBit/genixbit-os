# GenixBit OS Branding & Visual Identity Guidelines

This document outlines the visual identity, naming rules, artwork specifications, and trademark guidelines for **GenixBit OS**.

---

## Product Identity & Naming Standards

### Official Product Names
- **Full Display Name**: `GenixBit OS` *(Used in documentation, marketing, desktop branding, and user-facing text)*
- **Internal / Package Name**: `genixbitos` *(Lowercase, no spaces or special characters; used in code, scripts, hostname, and package identifiers)*
- **Space-Free Business String**: `GenixBitOS` *(Used in environment variables where spaces are prohibited by build scripts)*

### Company & Attribution
- **Company Name**: `GenixBit Labs Private Limited`
- **Official Website**: `https://www.genixbit.com`
- **OS Website**: `https://os.genixbit.com`
- **Documentation**: `https://docs.os.genixbit.com`
- **Package Repository**: `https://packages.os.genixbit.com`

---

## Visual Branding Assets Status

### Current Status (`0.1.0-alpha`)
* Core identity variables in build scripts have been updated to `GenixBitOS` and `genixbitos`.
* Vector graphic assets (`logo.svg`) and default desktop wallpapers are currently in a **transitional state**.
* Official GenixBit OS branded visual assets (vector logo, dark/light desktop wallpapers, Plymouth boot animation, GNOME icon themes) are scheduled for introduction in **Phase 2 (Milestone 0.2.0)**.

### Trademark & Logo Usage Guidelines
1. **GenixBit Trademarks**: The name "GenixBit", "GenixBit OS", "GenixBit Labs", and official logos belong to GenixBit Labs Private Limited.
2. **Upstream Trademarks**: Upstream project trademarks (Ubuntu, Canonical, AnduinOS) remain the property of their respective trademark holders and must be respected in all public documentation and distributions.
